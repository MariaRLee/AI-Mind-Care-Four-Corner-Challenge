AI-Mind Care｜四角號碼漢字挑戰 v1.0

使用方式：
1. 直接以瀏覽器開啟 index.html 測試。
2. 上傳 index.html 到新的 GitHub repository 根目錄。
3. 在 Settings → Pages 選擇 Deploy from a branch、main、/(root)。

特色：
- 中文／English／中英雙語
- 四角號碼0–9入門教學
- 5題與10題挑戰
- 拼音與英文意思
- 即時答案與四角位置提示
- 成績百分比、80%以上彩帶、100%煙火
- 手機與電腦版自適應
- 不使用訪客計數、Cookie、分析或外部追蹤
- 所有程式都在單一 index.html 內
