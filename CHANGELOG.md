# Changelog

## V2.0

- Expanded from 53 to 700 unique characters
- Added Entry, Basic, and Advanced levels
- Entry: 27 characters / 5-question quiz
- Basic: 200-character cumulative pool / 10-question quiz
- Advanced: 500 characters / 20-question quiz
- Added searchable and filterable character library
- Added 24-card pagination
- Added dynamic level and total counts
- Improved PC and mobile layouts
- Mobile tutorial remains collapsed until requested
- PC does not show the unnecessary Quick Guide start button
- Check button changes into Next after answering
- New entries without a level automatically enter Advanced
